# Personal AI Blog

Nuxt 4 + Vue 3 + TypeScript personal blog and portfolio site.

## Run

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Build

```bash
npm run build
npm run preview
```

## Deploy

This project can be deployed to Vercel, Netlify, Cloudflare Pages, or a Node server. For Vercel, import the repository and use the default Nuxt build settings.
